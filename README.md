# Smart-Expense-Guardian-An-AI-finance-analyzer
The Smart Expense Guardian is an intelligent financial management system that integrates machine learning techniques to
analyse and optimize personal spending behaviour. Unlike traditional expense tracking applications, this system not only
records transactions but also provides predictive insights and anomaly detection to enhance financial decision-making.
The application utilizes Natural Language Processing (NLP) to automatically categorize expenses based on user-provided
descriptions. An anomaly detection model identifies unusual spending patterns and alerts users to potential financial risks.
A regression-based predictive model forecasts future expenses, enabling users to plan budgets effectively.
The system presents data through interactive dashboards with pie charts and bar graphs. A Financial Health Score evaluates
the user's spending habits. An AI-powered chatbot delivers personalized financial insights in real time — transforming
conventional expense tracking into a smart, data-driven financial assistant.
